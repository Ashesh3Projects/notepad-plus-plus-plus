What is Notepad+++?
*******************

Notepad+++ is a free (as in "free speech" and also as in "free beer") source code editor and Notepad replacement that supports several programming languages and natural languages. Running in the MS Windows environment, its use is governed by GPL (GNU General Public License).


Why another source code editor?
*******************************

To provide a high-performance, lightweight text editor with enhanced features for developers and users alike. Notepad+++ is built with performance in mind, leveraging the Scintilla editing component to deliver fast and responsive editing capabilities.


How to install:
***************

From the installer:
	Just follow the installation instructions.
From the zip/7z package:
	Unzip all the files into a directory you want then launch "notepad++.exe".
